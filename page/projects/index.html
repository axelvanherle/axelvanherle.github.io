P2P-TCP-chatclient-NP

Found here.

A peer-to-peer chat client in C++ using object-oriented programming principles and QT. The chat client allows users to connect with each other using TCP protocol, communicate through sending and receiving messages, and receive a list of connected peers. The chat client is designed to be extensible, with new peers being added dynamically, and with the ability to send and receive messages to and from all connected peers. I made a blog post explaining it, which you can find here.

imgConvolutionCuda-C

Found here.

As part of the Hardware Accelerated Computing course, I completed an assignment that involved reading a PNG image, converting it to grayscale, applying a kernel, and performing max and min pooling on the image using C. Following this, I repeated the process using CUDA, which significantly improved the performance of the program. Through this project, I gained a great deal of experience with CUDA and parallelizing applications.

ticTacToeInCPP_task1
