How i got this site up and running.

Hello, so this post is just a quick explanation of how I got this website up and running.

First of all, it’s hosted via GitHub pages. This makes it so I don’t have to go through any effort of setting this up myself, and it’s nice practice in case I need to use it some other time. Although an argument could be made that hosting my own site would have been better practice.

Once I could actually host something, I decided that it’s time to make this something good. One small issue, I know nothing of web design. So, what’s the logical next step? Learning about it.

At one point, I had my own website online, made it using Bootstrap. This, however, isn’t a good option because to have something decent it would require me to actually learn these languages. So, I opted to learn something easier, Hugo.

Now, what is that you might ask? This site was built using it. It’s a static site generator, all I need to do is configure it and give it some Markdown files (like the one you are reading right now). I got most of what I know from watching this video, and the rest I learned from all other available documentation online.

TLDR; This is built using Hugo and hosted using GitHub pages.
